ABOUT ME

I love to build things.

Ever since I was little I've always been interested in creating things. Over the years my drive for creation manifested itself in a lot of different ways. I've experimented with drawing, painting, costume design, woodworking, stained glass, 3D printing and a myriad of other media. In high school I discovered the world of coding and created my first webite on Angelfire. It was HTML with inline CSS lifted from other sites and modified to my liking. It was terrible but I loved it. I've had the coding bug ever since and continued to maintain a website but it wasn't until recently that I started to consider it for a career.

PORTFOLIO

ProPlan

I love to build things but I get frustrated with the way my data ends up being scattered over several different mediums. For any given project I have sketches, handwritten notes and lists, links to supplies and references on my computer and progress photos on my phone. ProPlan was created as a way to bring all this information together so it can be easily referred back to. ProPlan can help the user track tasks, purchases, references and supplies all from their smartphone. I built ProPlan using React.js with React Bootstrap for styling and React-Icons.
